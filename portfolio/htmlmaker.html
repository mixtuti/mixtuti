HTML作成ツール

サークルサイトが簡単に作れる


制作期間 2日(一部制作中)
制作人数 1人
使用ツール Visual Studio Code
使用言語 HTML,CSS,JavaScript
リリース日 2023/06/04

概要

このツールは、代表を務める「Cadence」というサークルのサイトで使用するサイトのパーツやページを簡単に作れるようにするツールです。

入力ボックスに入力し「変換」ボタンをそうと下のテキストボックスにhtmlが出力されます。

「コピー」ボタンを押すことでクリップボードにコピーすることも可能です。

こちらから、作成ページに飛ぶことができます。(現在左の4種類作成済み)
